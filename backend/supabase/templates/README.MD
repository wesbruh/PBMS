## The html files in this folder strictly serve as the html templates that are used for Supabase Authentication. There is no direct connection within the codebase (but can be configed) are purely for reference. To actually modify the templates, go to 'Authentication' in the PBMS Supabase project. I had to enable these manually and easier to do this way.

### There are templates for:
- 'Confirm sign up' email
- 'Change email address' confirmation email (auth verify email change) 
- 'Password changed' confirmation email
- 'Email address changed' confirmation email
- 'Phone number changed' confirmation email
